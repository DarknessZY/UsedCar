## 二手汽车交易小程序（不涉及支付）之前自己学完vue2整的毕设

### 需求背景
### 技术栈
vue2+uni-app +vuex+colorUI+uni-ui
### 目录结构
```bash
pages/
!--- components 公共组件
!--- my 我的页面
!-- -home 首页 
!-- -issue 二手车发布页面
-- -disconver 二手车筛选购买页面
```
### 代码规范和接口
当时，因刚学习完就开始，没啥代码规范，接口也没封装直接用uni.request原生
### 本地开发
在hbulidX中打开项目，右上角运行运行到微信开发者工具中
